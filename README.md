1️⃣ What is the difference between var, let, and const?
Ans: var let and const are used to declare variables in JavaScript 
var is the older way and it is function-scoped meaning it works inside a function and can be redeclared and updated
let is block scoped so it only works inside the block where it is declared and it can be updated but not redeclared in the same scope
const is also block scoped like let but its value cannot be changed or redeclared after it is assigned.

2️⃣ What is the spread operator (...)?
Ans: The spread operator (...) is used to expand or copy elements of an array or object
it allows developers to combine arrays copy arrays or add new elements easily
for example, an array can be copied or merged with another array using the spread operator
making the code shorter and easier to read.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: map() filter() and forEach() are JavaScript array methods used to work with array elements map() creates a new array by applying a function to each element
 filter() creates a new array containing only the elements that meet a specific condition 
 forEach() simply runs a function for each element in the array but does not return a new array.

4️⃣ What is an arrow function?
Ans: An arrow function is a shorter and modern way to write functions in 
JavaScript using the => symbol it makes the code more concise and easier 
to read arrow functions are commonly used for simple functions and callback especially when working with array methods like map() and filter().

5️⃣ What are template literals?
Ans: template literals are a way to create strings in javaScript using backticks (`) instead of quotes they allow developers 
to easily insert variables and expressions inside a string using ${} this makes it easier to create dynamic strings and multi line text.
